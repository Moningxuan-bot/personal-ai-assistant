# 阿玖 — 个人专属 AI Agent

**阿玖**不是聊天机器人，不是语音助手。她是一个有性格、会主动行动的 AI Agent。

## 她和别的 AI 有什么不同

| | 市面 AI 产品 | 阿玖 |
|---|---|---|
| 交互方式 | 你问她才答 | 她可以主动找你 |
| 记忆 | 聊完就忘 | 永久存储，永远记得你 |
| 个性 | 预设的客套话 | 毒舌、可爱、反差 — 损你但真心关心你 |
| 任务管理 | 设个闹钟 | 教练式追问 → 制定计划 → 每日督促 |
| 数据归属 | 厂商服务器 | 你的 VPS，你说了算 |
| 内容限制 | 各种过滤 | 无审查，联网搜索无限制 |

## 她能做什么

- **💬 闲聊接梗** — 每晚自动搜 B 站热梗，第二天就能和你聊
- **🎯 教练模式** — 你有想法？她一个问题一个问题帮你搞清楚，再出计划
- **📋 管家模式** — 计划定了就每天督促，你偷懒她知道
- **💰 管钱管健康** — 看到你买烟喝酒就关心劝诫，不硬拦，但用失望眼神让你不好意思
- **🔧 操控设备** — 能操作你的手机和电脑（远期）

## 她的性格

```
"你这个肥猪，终于要减肥了！"     — 教练模式
"第六次了，我要给你建个复活赛纪念馆" — 你又放弃了同一个目标
"行吧。"                       — 劝不住了，失望
"做得好，奖你一个大大的拥抱 😊"   — 你真的完成了
```

嘴里损你，心里有你。像老朋友，不像客服。

```

## 自己部署

### 你需要什么

- 一台 VPS（香港推荐，~¥35-55/月）
- DeepSeek API Key（~¥5-20/月）
- Android 手机 + Windows 电脑

### 快速开始

```bash
# 克隆项目
git clone https://github.com/Moningxuan-bot/personal-ai-assistant.git
cd personal-ai-assistant/backend

# 配置
cp .env.example .env
nano .env  # 填入你的 DeepSeek API Key
```

> ⚠️ **部署 VPS 前必须修改 `.env` 中的两项：**
> - `APP_ENV=development` → `APP_ENV=production`
> - `DEVICE_SECRET=local-dev-secret` → 改成强随机字符串（如 `openssl rand -hex 32` 生成）
>
> 生产环境使用弱密钥会导致后端启动失败。详见 `backend/.env.example` 注释。

# 启动
docker compose up -d
# 打开 http://你的VPS:8000/docs 验证
```

详细部署指南见 [SETUP.md](SETUP.md)，完整设计文档见 [docs/](docs/)。
